Introduction to HTML

HTML (HyperText Markup Language) is the code that is used to structure a webpage
it's content. For example, content could be structed within a set of paragraphs, a list of
bulleted points or using images and data tables.

PS: It was invented by Physicist Tim Berners-Lee, the first version was published in 1993.

So what is HTML?

HTML is a markup language that defines the structure of your content. It consists of a
series of elements which you use to enclose, or wrap different parts of the content to make it
appear or act a certain way. For example, if we wanted to a line to stand by itself, we could
